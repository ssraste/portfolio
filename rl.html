Object Manipulation in Musculoskeletal Simulations

Reinforcement Learning MuJoco Myosuite

  • Inspired by MyoChallenge 2023, I trained a virtual musculoskeletal arm model to grasp, lift, and place an object into a designated target box using MuJoco simulator and MyoSuite's Relocation environment to perform pick and place task.

  • I used Proximal Policy Optimization algorithm as it provides stable training in continuous control tasks.

  • MyoArm model has 27 DoFs and 63 muscle-tendon units, the agent initially explores all possible states

  • I tried using MyoSuite's default reward structure, but then optimized it with Custom intermediate rewards and penalties to improve the training time and fine-tuned Hyperparameter to get higher and stable rewards.

    • Proximal Policy Optimization

  • The agent interacts with an environment by taking actions based on its current state. After each action, the environment provides feedback in the form of a reward, which tells the agent how good or bad that action was for achieving its goal. The agent’s goal is to learn a policy that maximizes the total reward over time.

  • I used PPO as it was simpler to implement and more stable during training. The downside is that it took more training steps to reach the goal compared to SAC.

    • Image by alaminhnab4 on Medium

    Training with Default Settings

  • Training with default reward structure and PPO hyperparameters, the agent explored all possible states which leads to low rewards and increased training time.

  • The value loss decreases slowly, while the policy loss remains erratic and stabilizes gradually.

  • The reward curve shows slow improvement as the overall rewards are improving but still negative due to the complexity of controlling many DOFs.

  • Occasionally, there are spikes in negative rewards as the agent explores new states.

    • Training with Custom Settings

  • Introduced a custom reward structure to provide intermediate feedback, which encouraged desired behaviors and Penalized undesired actions and also fine-tuned the hyperparameters for stable updates.

  • The agent attempts to grasp the object by applying torques to its joints; however, it ends up pushing the object farther away, suggesting that more training is needed.

  • There is a decrease in the Value loss curve is much smoother compared to the earlier plots.

  • The Policy loss curve is stable and gradually decreased, which shows improvement in training and the rewards curve shows positive rewards.
